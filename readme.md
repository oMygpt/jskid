# 🚀 AMAC 光速浏览器 - 挂机助手使用指南 (v3.0)

本脚本专为 **AMAC (中国基金业协会) 远程培训系统** 设计，采用 **v3.0 [事件大师]** 引擎，通过毫秒级 DOM 监听技术，实现全自动、高效率、高确认率的挂机体验。

## 🌟 核心功能

*   **⚡ 瞬移倍速 (Stealth Warp)**：采用 12x 物理加速 + 每秒 3.5s 进度跳跃的组合算法，实测综合效率可达 **20倍速以上**，且能稳定绕过服务器的时间校验。
*   **🛡️ 25秒原速结算区**：视频播放到最后 25 秒时，自动降速至 **1.0x 绝对原速**。这不仅仅是为了心跳包，更是为了触发系统关键的“评价/反馈”对话框。
*   **🤖 毫秒级弹窗监听**：基于 `MutationObserver` 技术，脚本能瞬间捕获页面上弹出的任何对话框。无论是中途的暂停提示，还是结尾的**评价确认框**，都能在露头的一刹那被自动点掉。
*   **🚫 深度防暂停**：即使浏览器失去焦点、切换标签页或最小化，视频依然保持运行，不再弹出“请勿离开页面”的限制。
*   **🔒 状态强锁定**：视频一旦播完立即锁定 `ended` 状态，彻底杜绝视频循环播放或重头再来的问题。

## 🛠️ 安装与部署

### 1. 配合“AMAC光速浏览器”使用（推荐）
如果您正在使用我们开发的定制 Electron 浏览器：
- 启动程序，脚本已内置在浏览器内核中。
- 直接登录 `peixun.amac.org.cn` 即可。

### 2. 通过 Tampermonkey (油猴) 手动安装
如果您在 Chrome、Edge 或 Safari 中使用：
1. 安装 [Tampermonkey 扩展程序](https://www.tampermonkey.net/)。
2. 点击油猴图标 -> “添加新脚本”。
3. 将项目中的 `amac_god_mode.user.js` 代码全部复制并粘贴进去。
4. 按 `Ctrl+S` (或 `Cmd+S`) 保存并刷新页面。

## 📖 使用说明

1.  **进入课程**：登录平台并打开任意视频课程页面。
2.  **脚本激活**：控制台（F12 -> Console）显示 `--- AMAC GOD MODE v3.0 [EVENT MASTER] ACTIVATED ---` 即表示成功。
3.  **全自动流程**：
    *   视频会自动开启“瞬移模式”，进度条会以非线性方式快速推进。
    *   **关键点**：当视频剩余时间少于 25 秒时，进度条会恢复正常语速，请耐心等待系统弹出“评价框”或“确认框”，脚本会自动处理。
4.  **自动过关**：脚本在确认评价框已处理且视频彻底结束后，会自动尝试跳转到下一节或测验页。

---

## ⚠️ 重要注意事项（必读）

*   **不要干扰结算区**：视频最后 25 秒是服务器确认进度的核心阶段。在此期间请勿手动拖动进度条，否则可能导致确认失败。
*   **关于评价框**：某些课程必须点击“满意度评价”后服务器才会标记完成。v3.0 已实现自动识别，请确保脚本有权限操作 DOM。
*   **禁止多开**：请确保**一次只在一个标签页**播放视频。多开可能导致服务器端数据冲突。

## ❓ 常见问题 (FAQ)

**Q: 为什么视频快结束时突然变慢了？**  
A: 这是为了触发系统的“仪式感对话框”（评价/反馈）。没有这一步，服务器往往不会确认进度。

**Q: 脚本显示视频播完了，但没自动跳转？**  
A: 脚本在确认所有弹窗处理完毕后，有 5-15 秒的安全等待期。如果长时间不跳转，请手动点击“下一节”。

**Q: 脚本会被后台检测到吗？**  
A: v3.0 采用了模拟人类行为的“原速结算”和“非线性瞬移”，目前在 AMAC 平台上是非常安全的。
